# Documentation Roadmap (XAMPP/MAMP)

Dokumentasi aktif sekarang berfokus pada setup manual non-container.

## Urutan Baca

1. XAMPP_QUICK_START.md

- Fast path untuk menjalankan aplikasi lokal.

2. XAMPP_SETUP.md

- Setup detail server, virtual host, database, dan konfigurasi Yii.

3. XAMPP_FACTORY_CHECKLIST.md

- Validasi akhir sebelum dipakai operasional.

4. NETWORK_SECURITY.md

- Hardening jaringan pabrik dan kontrol akses.

5. API_FORM_SUBMISSION.md dan API_TESTING.md

- Integrasi Android dan validasi endpoint.

6. RBAC_QUICKSTART.md

- Setup role dan permission.

## Scope

- Didukung: XAMPP/MAMP.
- Tidak digunakan: workflow container.
