# Setup Choice Guide

## Keputusan Final

Gunakan XAMPP/MAMP untuk project ini.

## Kenapa

- Lingkungan saat ini sudah berjalan di path MAMP.
- Konfigurasi database lokal sudah disiapkan untuk jalur non-container.
- Tim tidak memakai workflow container untuk operasional project ini.

## Langkah Berikut

1. Ikuti XAMPP_QUICK_START.md untuk startup cepat.
2. Ikuti XAMPP_SETUP.md untuk konfigurasi lengkap.
3. Jalankan migration dan inisialisasi RBAC.
4. Lanjutkan ke API testing.
